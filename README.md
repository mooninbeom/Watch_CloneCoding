# 시계 클론 코딩
아이폰 기본 앱에 포함되어 있는 **시계** 어플을 클론 코딩 해보았습니다.

**UIKit**을 사용하여 개발했습니다.


* 세계 시계
* 알람
* 스톱워치
* 타이머

기능이 포함되어 있습니다.
